ROT15 Encryption Algorithm with Salting

This Python script implements a text encryption and decryption program using a modified ROT15 encryption technique with added salting. It utilizes the tkinter library to create a graphical user interface (GUI) for user interaction.

Introduction

The script offers a secure text encryption and decryption solution. It employs a customized version of the ROT15 encryption method, enhanced with a salting mechanism. The graphical user interface makes it accessible and user-friendly, allowing users to encrypt and decrypt text with ease. Additionally, random keys are generated for each encryption operation, enhancing security.

Functions

clear_text(text_widget)

This function clears the content of a tkinter.Text widget, ensuring a clean slate for further text input.

Parameters:

  • text_widget: The tkinter.Text widget that will be cleared.

encrypt_text(input_text, output_text)

The encrypt_text function handles the encryption process using a custom ROT15-based algorithm with added salting for enhanced security.

Parameters:

  • input_text: The input tkinter.Text widget where the user enters the text to be encrypted.
  • output_text: The tkinter.Text widget that displays the encrypted text and combined keys.

decrypt_text(input_text, keys_entry, output_text)

The decrypt_text function manages the decryption process, allowing users to enter the encrypted text and the combined keys to decrypt it successfully.

Parameters:

  • input_text: The input tkinter.Text widget for the encrypted text.
  • keys_entry: The tkinter.Entry widget where the user inputs the combined keys in the format key1|key2.
  • output_text: The tkinter.Text widget that displays the decrypted text.

get_random_int(min_value, max_value)

The get_random_int function generates a random integer between specified minimum and maximum values, providing the essential randomness needed for the encryption keys.

Parameters:

  • min_value: The minimum value for the random integer.
  • max_value: The maximum value for the random integer.

Main Function

The main function is the entry point of the program, setting up the graphical user interface (GUI) with the help of the tkinter library. It creates a window with two tabs for encryption and decryption, enabling user interaction.

User Interface

Encryption Tab

The Encryption Tab allows users to input text for encryption and initiate the encryption process. Key features include:

  • Label 1: A label prompting users to "Enter the text to encrypt."
  • Input Text Box 1: A tkinter.Text widget where users input the text to be encrypted.
  • Encrypt Button: A button labeled "Encrypt" that triggers the encryption process. It calls the encrypt_text function.
  • Output Text Box: A tkinter.Text widget that displays the encrypted text and combined keys after encryption is complete.

Decryption Tab

The Decryption Tab is used for decrypting text encrypted with the program. Key features include:

  • Label 2: A label instructing users to "Enter the encrypted text."
  • Input Text Box 2: A tkinter.Text widget where users input the encrypted text.
  • Label 3: A label guiding users to "Enter the combined keys (key1|key2)."
  • Keys Entry: A tkinter.Entry widget for inputting the combined keys in the format key1|key2.
  • Decrypt Button: A button labeled "Decrypt" that initiates the decryption process. It calls the decrypt_text function.
  • Output Text Box: A tkinter.Text widget that displays the decrypted text after decryption is complete.

Execution

The script is designed to be executed as a standalone program. When run, it launches a graphical user interface that enables users to encrypt and decrypt text. The program employs a custom ROT15 encryption method with additional salting to enhance text security. Additionally, random keys are generated for each encryption, making it challenging to decrypt without the proper keys.

Note: Ensure that the necessary libraries, including tkinter and random, are installed in your Python environment before running the script. This program is an excellent choice for users who require a user-friendly, secure text encryption and decryption solution.

Este script Python implementa um programa de criptografia e descriptografia de texto usando uma técnica de criptografia ROT15 modificada com a adição de salmoura. Ele utiliza a biblioteca tkinter para criar uma interface gráfica de usuário (GUI) para interação do usuário.

Introdução

O script oferece uma solução segura de criptografia e descriptografia de texto. Ele emprega uma versão personalizada do método de criptografia ROT15, aprimorada com um mecanismo de salmoura. A interface gráfica de usuário torna acessível e amigável, permitindo que os usuários criptografem e descriptografem texto com facilidade. Além disso, chaves aleatórias são geradas para cada operação de criptografia, aprimorando a segurança.

Funções

clear_text(text_widget)

Esta função limpa o conteúdo de um widget tkinter.Text, garantindo uma tela limpa para mais entradas de texto.

Parâmetros:

  • text_widget: O widget tkinter.Text que será limpo.

encrypt_text(input_text, output_text)

A função encrypt_text lida com o processo de criptografia usando um algoritmo personalizado baseado em ROT15 com salmoura adicionada para maior segurança.

Parâmetros:

  • input_text: O widget tkinter.Text de entrada onde o usuário insere o texto a ser criptografado.
  • output_text: O widget tkinter.Text que exibe o texto criptografado e as chaves combinadas.

decrypt_text(input_text, keys_entry, output_text)

A função decrypt_text gerencia o processo de descriptografia, permitindo que os usuários insiram o texto criptografado e as chaves combinadas para descriptografá-lo com sucesso.

Parâmetros:

  • input_text: O widget tkinter.Text de entrada para o texto criptografado.
  • keys_entry: O widget tkinter.Entry onde o usuário insere as chaves combinadas no formato chave1|chave2.
  • output_text: O widget tkinter.Text que exibe o texto descriptografado.

get_random_int(min_value, max_value)

A função get_random_int gera um número inteiro aleatório entre valores mínimos e máximos especificados, fornecendo a aleatoriedade essencial necessária para as chaves de criptografia.

Parâmetros:

  • min_value: O valor mínimo para o número inteiro aleatório.
  • max_value: O valor máximo para o número inteiro aleatório.

Função Principal

A função principal é o ponto de entrada do programa, configurando a interface gráfica de usuário (GUI) com a ajuda da biblioteca tkinter. Ela cria uma janela com duas guias para criptografia e descriptografia, permitindo a interação do usuário.

Interface do Usuário

Guia de Criptografia

A Guia de Criptografia permite que os usuários insiram o texto para criptografia e iniciem o processo de criptografia. Recursos-chave incluem:

  • Rótulo 1: Um rótulo que instrui os usuários a "Insira o texto a ser criptografado."
  • Campo de Texto de Entrada 1: Um widget tkinter.Text onde os usuários inserem o texto a ser criptografado.
  • Botão Criptografar: Um botão rotulado como "Criptografar" que aciona o processo de criptografia. Ele chama a função encrypt_text.
  • Campo de Texto de Saída: Um widget tkinter.Text que exibe o texto criptografado e as chaves combinadas após a conclusão da criptografia.

Guia de Descriptografia

A Guia de Descriptografia é usada para descriptografar o texto criptografado com o programa. Recursos-chave incluem:

  • Rótulo 2: Um rótulo que instrui os usuários a "Insira o texto criptografado."
  • Campo de Texto de Entrada 2: Um widget tkinter.Text onde os usuários inserem o texto criptografado.
  • Rótulo 3: Um rótulo que orienta os usuários a "Insira as chaves combinadas (chave1|chave2)."
  • Entrada de Chaves: Um widget tkinter.Entry para inserir as chaves combinadas no formato chave1|chave2.
  • Botão Descriptografar: Um botão rotulado como "Descriptografar" que inicia o processo de descriptografia. Ele chama a função decrypt_text.

Execução

O script é projetado para ser executado como um programa independente. Quando executado, ele inicia uma interface gráfica de usuário que permite aos usuários criptografar e descriptografar texto. O programa utiliza um método de criptografia ROT15 personalizado com adição de salmoura para aprimorar a segurança do texto. Além disso, chaves aleatórias são geradas para cada operação de criptografia, tornando desafiador descriptografar sem as chaves corretas.

Observação: Certifique-se de que as bibliotecas necessárias, incluindo tkinter e random, estejam instaladas em seu ambiente Python antes de executar o script. ROT15-Verschlüsselungsalgorithmus mit Salzen

Dieses Python-Skript implementiert ein Textverschlüsselungs- und -entschlüsselungsprogramm mit einer modifizierten ROT15-Verschlüsselungstechnik und zusätzlichem Salzen. Es verwendet die tkinter-Bibliothek, um eine grafische Benutzeroberfläche (GUI) für die Benutzerinteraktion zu erstellen.

Einführung

Das Skript bietet eine sichere Lösung zur Verschlüsselung und Entschlüsselung von Text. Es verwendet eine angepasste Version der ROT15-Verschlüsselungsmethode, die mit einem Salzmechanismus erweitert wurde. Die grafische Benutzeroberfläche macht sie zugänglich und benutzerfreundlich, so dass Benutzer Texte einfach verschlüsseln und entschlüsseln können. Darüber hinaus werden zufällige Schlüssel für jede Verschlüsselungsoperation generiert, um die Sicherheit zu erhöhen.

Funktionen

clear_text(text_widget)

Diese Funktion löscht den Inhalt eines tkinter.Text-Widgets und sorgt für eine saubere Oberfläche für weitere Texteingaben.

Parameter:

  • text_widget: Das tkinter.Text-Widget, das gelöscht wird.

encrypt_text(input_text, output_text)

Die Funktion encrypt_text behandelt den Verschlüsselungsprozess unter Verwendung eines benutzerdefinierten ROT15-basierten Algorithmus mit zusätzlichem Salzen für erhöhte Sicherheit.

Parameter:

  • input_text: Das Eingabe-Text-Widget von tkinter, in dem der Benutzer den zu verschlüsselnden Text eingibt.
  • output_text: Das tkinter.Text-Widget, das den verschlüsselten Text und die kombinierten Schlüssel anzeigt.

decrypt_text(input_text, keys_entry, output_text)

Die Funktion decrypt_text verwaltet den Entschlüsselungsprozess und ermöglicht es den Benutzern, den verschlüsselten Text und die kombinierten Schlüssel erfolgreich zu entschlüsseln.

Parameter:

  • input_text: Das Eingabe-Text-Widget von tkinter für den verschlüsselten Text.
  • keys_entry: Das tkinter.Entry-Widget, in dem der Benutzer die kombinierten Schlüssel im Format Schlüssel1|Schlüssel2 eingibt.
  • output_text: Das tkinter.Text-Widget, das den entschlüsselten Text anzeigt.

get_random_int(min_value, max_value)

Die Funktion get_random_int generiert eine zufällige Ganzzahl zwischen den angegebenen Mindest- und Höchstwerten und sorgt so für die erforderliche Zufälligkeit der Verschlüsselungsschlüssel.

Parameter:

  • min_value: Der Mindestwert für die zufällige Ganzzahl.
  • max_value: Der Höchstwert für die zufällige Ganzzahl.

Hauptfunktion

Die Hauptfunktion ist der Einstiegspunkt des Programms und richtet die grafische Benutzeroberfläche (GUI) mit Hilfe der tkinter-Bibliothek ein. Sie erstellt ein Fenster mit zwei Registerkarten für Verschlüsselung und Entschlüsselung und ermöglicht die Benutzerinteraktion.

Benutzeroberfläche

Verschlüsselung

Die Registerkarte "Verschlüsselung" ermöglicht es den Benutzern, Text für die Verschlüsselung einzugeben und den Verschlüsselungsprozess zu starten. Zu den wichtigsten Funktionen gehören:

  • Label 1: Ein Label, das die Benutzer auffordert, "Den zu verschlüsselnden Text einzugeben."
  • Eingabe-Textfeld 1: Ein tkinter.Text-Widget, in dem die Benutzer den zu verschlüsselnden Text eingeben.
  • Verschlüsseln-Schaltfläche: Eine Schaltfläche mit der Aufschrift "Verschlüsseln", die den Verschlüsselungsprozess auslöst. Sie ruft die encrypt_text-Funktion auf.
  • Ausgabe-Textfeld: Ein tkinter.Text-Widget, das den verschlüsselten Text und die kombinierten Schlüssel nach Abschluss der Verschlüsselung anzeigt.

Entschlüsselung

Die Registerkarte "Entschlüsselung" wird für die Entschlüsselung von mit dem Programm verschlüsseltem Text verwendet. Zu den wichtigsten Funktionen gehören:

  • Label 2: Ein Label, das die Benutzer anweist, "Den verschlüsselten Text einzugeben."
  • Eingabe-Textfeld 2: Ein tkinter.Text-Widget, in dem die Benutzer den verschlüsselten Text eingeben.
  • Label 3: Ein Label, das die Benutzer anleitet, "Die kombinierten Schlüssel (Schlüssel1|Schlüssel2) einzugeben."
  • Schlüsseleingabe: Ein tkinter.Entry-Widget zum Eingeben der kombinierten Schlüssel im Format Schlüssel1|Schlüssel2.
  • Entschlüsseln-Schaltfläche: Eine Schaltfläche mit der Aufschrift "Entschlüsseln", die den Entschlüsselungsprozess startet. Sie ruft die decrypt_text-Funktion auf.
  • Ausgabe-Textfeld: Ein tkinter.Text-Widget, das den entschlüsselten Text nach Abschluss der Entschlüsselung anzeigt.

Ausführung

Das Skript ist darauf ausgelegt, als eigenständiges Programm ausgeführt zu werden. Beim Start wird eine grafische Benutzeroberfläche gestartet, die es den Benutzern ermöglicht, Text zu verschlüsseln und zu entschlüsseln. Das Programm verwendet eine benutzerdefinierte ROT15-Verschlüsselungsmethode mit zusätzlichem Salzen, um die Sicherheit des Textes zu erhöhen. Darüber hinaus werden zufällige Schlüssel für jede Verschlüsselungsoperation generiert, was die Entschlüsselung ohne die richtigen Schlüssel erschwert.

Hinweis: Stellen Sie sicher, dass die erforderlichen Bibliotheken, einschließlich tkinter und random, in Ihrer Python-Umgebung installiert sind, bevor Sie das Skript ausführen. Dieses Programm ist eine hervorragende Wahl für Benutzer, die eine benutzerfreundliche und sichere Lösung zur Verschlüsselung und Entschlüsselung von Text benötigen.
